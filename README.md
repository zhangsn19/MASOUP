# SOUPS26

This repo contains the descriptions for the paper, in the following manner:

- classification.md : The classification of private information. 
- codebook.md : The codebook for the evaluation study.
- design_all.md : The design for different settings, including descriptions on how users could interact with the corresponding settings.
- exit_interview.md : The outline of the exit interview.
- implementation_eval.md : The implementation details of the system in the evaluation study.
- implementation_proactive.md : The implementation details of the proactive setting in the evaluation study.
